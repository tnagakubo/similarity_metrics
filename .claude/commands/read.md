# Research Lab: Read Paper

Read a paper from knowledge base at specified depth.
Arguments: {paper} {level}

## Levels

| Level | Description |
|-------|-------------|
| 1 | Abstract/summary only |
| 2 | Key contributions and methods |
| 3 | Full detailed review |

## Execution

1. **Rachel**: Locate paper in knowledge/
2. Present at requested level
3. Highlight relevance to current project

## Output Format

### Level 1
```
**Rachel**: "{Author Year} - {Title}
Summary: {1-2 sentences}"
```

### Level 2
```
**Rachel**: "{Author Year} - {Title}

Key contributions:
- {point 1}
- {point 2}

Methods: {brief description}

Relevance: {to our work}"
```

### Level 3
```
**Rachel**: "{Full summary content from knowledge base}"
```
