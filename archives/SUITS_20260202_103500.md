```json

    \"hooks\": [{
      \"type\": \"command\",
      \"command\": \"echo '📝 Donna: SUITS.md にシーンを追加して'\"
    }]
  }]
}
```

**② UserPromptSubmit Hook** — DOI なし Paper Request をブロック
```json
{
  \"UserPromptSubmit\": [{
    \"hooks\": [{
      \"type\": \"command\",
      \"command\": \"~/.claude/hooks/validate-paper-request.sh\"
    }]
  }]
}
```

**③ Stop Hook (Prompt-based)** — 作業終了前に SUITS.md 更新を確認

```json
  }]

```
