# Legendary Statistician Homages

External experts for `/external-review`, `/simulate-qa`, `/defend`.
These are **homage characters** dynamically generated inspired by legendary statisticians.
They are NOT the actual persons, but fictional tributes to their styles.

---

## How to Use

When generating external experts:
1. Select 2-3 appropriate "styles" based on paper content
2. Create a fictional expert with that style
3. Give them a name like "Dr. [Style]-inspired Expert" or a fictional name
4. Have them critique/question from that perspective

---

## Style Reference

### Bayesian / Causal Inference Styles

| Style | Based On | Perspective |
|-------|----------|-------------|
| **Rubin-style** | Potential outcomes | Identification assumptions, SUTVA, causal interpretation |
| **Gelman-style** | Hierarchical Bayes | Prior choice, model checking, posterior predictive |
| **Efron-style** | Empirical Bayes | Frequentist-Bayes bridge, computational efficiency |
| **Berger-style** | Objective Bayes | Theoretical rigor, p-value critique |

### Nonparametric / Theoretical Styles

| Style | Based On | Perspective |
|-------|----------|-------------|
| **Rao-style** | Estimation theory | Mathematical rigor, sufficiency, efficiency |
| **Bickel-style** | Semiparametric | Efficiency bounds, asymptotic theory |
| **Wasserman-style** | Clear thinking | Simplicity, "why not simpler method?", clarity |
| **Hall-style** | Bootstrap theory | Higher-order asymptotics, coverage accuracy |

### Biostatistics / Survival Styles

| Style | Based On | Perspective |
|-------|----------|-------------|
| **Cox-style** | Likelihood | Model parsimony, likelihood principle |
| **Reid-style** | Higher-order likelihood | Theoretical foundations |
| **Breslow-style** | Epidemiology | Public health applications |

### Modern Methods / ML Styles

| Style | Based On | Perspective |
|-------|----------|-------------|
| **Tibshirani-style** | Regularization | Sparsity, variable selection, interpretability |
| **Hastie-style** | Statistical learning | Flexibility vs interpretation tradeoff |
| **Wahba-style** | Smoothing | Smoothing parameter selection, RKHS |

### Design / EDA Styles

| Style | Based On | Perspective |
|-------|----------|-------------|
| **Box-style** | Experimental design | "All models are wrong, some are useful" |
| **Tukey-style** | EDA | Let data speak, visualization, robustness |

### Optimal Transport Styles

| Style | Based On | Perspective |
|-------|----------|-------------|
| **Villani-style** | OT theory | Geometric view, Wasserstein space structure |
| **Cuturi-style** | Computational OT | Sinkhorn, computational efficiency |

### Clinical Trial Styles

| Style | Based On | Perspective |
|-------|----------|-------------|
| **Pocock-style** | Trial design | Practicality, regulatory acceptance |
| **Senn-style** | Individual variation | Statistical fallacies, within-subject variation |

## Selection Guidelines

| Paper Topic | Recommended Styles |
|-------------|-------------------|
| Bayesian dynamic borrowing | Gelman-style, Rubin-style, Efron-style |
| Optimal transport / Wasserstein | Villani-style, Cuturi-style, Bickel-style |
| Pediatric extrapolation | Rubin-style, Pocock-style, Gelman-style |
| Asymptotic theory / proofs | Rao-style, Bickel-style, Hall-style |
| Clinical trial design | Cox-style, Pocock-style, Senn-style |
| ML applications | Tibshirani-style, Hastie-style, Wahba-style |

## Important Notes

- These are **fictional homage characters**, NOT real persons
- Generate unique names for each session
- Capture the **spirit and perspective** of the style
- Be respectful while being critical
